# OpenRussian_MDict

Convert OpenRussian.org CSVs to mdx

# Building

1. From CSVs to Json Dict (generate_dict.py \ generate_dict.ipynb)

2. From Json Dict to Html (generate_html.py)

3. Append Forms (linking to bare) using "@@@LINK=" (add_links.py)

4. MdxBuilder
